# trivia


This repository hosts a multiplayer trivia project (Wits & Wagers style).

## Deploying to Vercel

See [VERCEL_DEPLOY.md](VERCEL_DEPLOY.md) for step-by-step deployment instructions.

Key points:
- The app lives in `wits-wagers-online/`.
- Environment variables must be set in the Vercel dashboard (see `.env.example`).
- The root `vercel.json` is configured for monorepo support.

The app lives in `wits-wagers-online/`.
